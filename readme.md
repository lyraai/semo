## 简介
这是一个基于 Expo 和 Next.js 构建的跨平台应用，支持移动端和 Web 端开发。该应用使用 Redux 来管理状态，Axios 处理 API 请求，React Navigation 处理页面路由。这个app是ai powered的疗愈app，目前的主要功能针对分手或者亲民关系后创伤提供的一系列疗愈

##
核心功能细化与增强：
专属AI疗愈助手
长期疗愈计划：AI可以为用户制定一个长期的情感疗愈计划，结合情感分析、课程推荐、人际关系管理等功能，打造个性化的“疗愈旅程”。

1.Healing Chat AI疗愈聊天
情感识别技术：除了基于文本的情感识别外，可以扩展为多模态，结合语音、面部表情等来更全面地评估用户的情绪。对话过程中的语气、言辞等信息都可以帮助AI更精准地调整回复方式。
深度对话引导：为用户设计多种对话模式，如情绪释放、压力管理、情感修复等，帮助用户引导情绪表达。
疗愈师风格：为用户不同疗愈师的预设，可以自定义，往后可与专业心理咨询师合作。
个性化对话建议：基于用户情感历史和偏好，AI可以提供高度个性化的疗愈建议，甚至定制每日任务或目标。
情绪急救：针对不对场景下的情绪急救，如分手。
2. Discover 疗愈课程
课程个性化推荐：AI可以根据用户当前的情绪状态、过往的行为模式或情绪波动历史，推荐合适的疗愈课程或活动。可以进一步细化课程种类，例如针对压力、孤独感、焦虑等具体情绪的疗愈方法。
情绪追踪与课程结合：用户在完成课程或活动后，可以记录情绪状态，AI根据变化给予反馈或分析，以帮助用户了解不同疗愈方式的效果。
3. MoodJourney 情绪记录
多模态情绪输入：允许用户通过文本、语音或照片（表情识别）记录情绪，从而更丰富地表达内心感受。
情绪趋势分析：AI可以自动生成情绪趋势图，帮助用户追踪和理解情绪变化，并在需要时提供建议，如冥想、呼吸练习或对话支持。
4. Healing kit 工具库
情感动态学习：除了静态资源（文章、音频、视频）外，AI可以结合用户的情绪状态实时推荐内容，如放松音频、心理学研究、情感管理技巧。
专属内容生成：根据用户的情绪和对话记录，生成特定的心理疗愈内容，类似于每天一篇自我反思或疗愈练习。
心理暗示卡牌：可以开发一套基于心理学的“暗示卡牌”，用户每天可以抽取一张卡牌，卡牌内容为正面引导、情绪支持或心理疗愈提示。可以与AI疗愈结合，用户根据抽到的卡片进行情感反思，AI也会根据反馈给予个性化建议。
每日挑战：通过每日的心理暗示卡片，用户可以获得“每日挑战”任务，如完成一次冥想、写下三件感谢的事情等，帮助用户构建积极的心理习惯。


AI关系管理与改善
人际关系诊断：用户可以输入与特定人际关系相关的信息（例如朋友、家人、伴侣等），AI可以通过情感分析和关系互动模式给出一些建议，帮助用户更好地理解和改善人际关系。
AI提供的沟通建议：在用户与他人有情绪摩擦或人际问题时，AI可以提供适合的沟通策略或回复建议，帮助用户在关系处理中更加冷静和理智。

情绪提醒与推送：根据用户的情绪波动，AI可以主动推送提醒，如“今天的情绪波动较大，建议进行冥想或深呼吸练习”，让用户随时得到情绪管理提示。
未来版本扩展：

青少年/儿童专版：面向青少年或儿童设计简化的功能版本，例如以游戏化的方式引导情感表达和管理，避免成人版中的复杂情感分析，更多关注轻松、互动的方式帮助情绪表达。
社交功能：创建安全、匿名的线上情感支持小组，让用户与其他人分享自己的感受和故事，互相支持。


4. Healing kit 工具库
1. 心情地图（Emotion Map）
描述：根据用户每天的情绪记录生成一张“心情地图”，显示情绪的地理分布（用户可以选择与某些特定地点的情绪关联）。
用途：帮助用户识别哪些环境或场景可能对情绪有积极或消极的影响，以更好地规划日常活动。
2. 回忆封存箱（Memory Vault）
描述：允许用户将与情感创伤相关的回忆、照片、文字或音频“封存”进一个数字箱子，标记某个日期后重新查看。
用途：通过主动处理回忆并决定暂时封存，帮助用户对过去的情感做一个阶段性告别。
3. 情绪语音记录（Emotion Voice Diary）
描述：通过语音记录用户的情绪状态，AI对语音情绪进行识别和分析，自动生成情绪报告。
用途：对于不喜欢打字的用户，语音记录可以更加自然且便利，同时提供情绪分析功能帮助用户反思和理解情绪。
4. 音乐疗愈（Music Therapy）
描述：根据用户的情绪状态和偏好生成一组疗愈音乐或播放列表，帮助用户放松或重新振作。
用途：音乐具有治愈和放松作用，通过AI推荐或生成个性化音乐，提高疗愈效果。
5. 疗愈时间线（Healing Timeline）
描述：以时间线的方式展示用户的疗愈旅程，包括情绪波动、完成的疗愈任务、重要里程碑等。
用途：让用户可视化他们的进步，增强持续使用的动力，看到疗愈过程中的小进步。
6. 虚拟倾诉伙伴（Virtual Companion）
描述：除了AI疗愈聊天之外，用户可以选择一个虚拟伙伴（如宠物或卡通角色）来分享日常感受，进行非正式的情感互动。
用途：通过虚拟伙伴提供一种轻松、不设防的情绪倾诉渠道，帮助用户保持积极的互动。
7. 随机慰藉卡片（Random Comfort Cards）
描述：用户在情绪低落时可以随机抽取一张慰藉卡片，内容包括鼓励的文字、名人名言或建议的行为（如喝杯热茶、去散步）。
用途：通过随机的积极心理学元素，帮助用户在特定时刻感到被理解和鼓励。
8. 释放练习（Letting Go Exercises）
描述：提供专门的练习和活动，帮助用户学习如何放下对过去的执念，如冥想、书写练习等。
用途：支持用户学会放下并接受情绪变化，逐步恢复心理健康。
9. 心理健康挑战（Healing Challenges）
描述：设置每日/每周挑战，例如“每天写三件感谢的事情”或“每日散步10分钟”，帮助用户养成积极的习惯。
用途：通过小的行为挑战逐步培养用户健康的心理和行为模式，增加疗愈过程中的成就感。
10. 智能情绪预测（Emotion Prediction）
描述：AI根据用户的情绪历史数据和行为模式，预测接下来可能的情绪波动，并提供相关的应对建议。
用途：让用户更好地了解自身情绪变化规律，预先做好情绪管理，减少情绪低落的影响。
11. 社交支持小组（Peer Support Group）
描述：用户可以选择加入匿名的在线支持小组，分享和讨论疗愈过程中的挑战和感受。
用途：通过群体的力量让用户感到支持，提供共同面对情感挑战的机会，帮助他们减少孤独感。
12. 智能冥想教练（Smart Meditation Coach）
描述：根据用户当前情绪状态生成定制化的冥想课程，如引导放松、情绪释放或正念冥想等。
用途：通过个性化的冥想练习，帮助用户有效地舒缓情绪，达到更好的疗愈效果。
13. 进步奖章系统（Achievement Badges）
描述：用户完成某些疗愈任务或达到特定目标后，会获得对应的奖章或奖励，如“首次情绪记录”、“坚持冥想七天”等。
用途：通过奖励机制，激励用户继续他们的疗愈旅程，并保持积极性。
14. 晨间提醒与夜间反思（Morning Affirmations & Nightly Reflections）
描述：每天早上推送正面的自我肯定语句，晚上提供反思问题帮助用户总结一天的情绪。
用途：通过日常的自我肯定和反思，帮助用户建立健康的心理习惯。

## 安装指南

### 1. 环境要求
确保你的开发环境中已经安装了以下工具：
- **Node.js**：版本 >= 16.x（兼容 Expo 和 Next.js）
- **Yarn**：建议使用 Yarn 来管理依赖。
- **Expo CLI**：可以使用 `npx expo` 来启动 Expo 开发环境。
- **Xcode**（仅限 iOS 开发）：如果你计划在 iOS 设备上进行开发，需要安装 [Xcode]

### 2. 安装步骤

#### 1. 克隆仓库

使用以下命令克隆项目仓库到本地：

```bash
git clone https://github.com/lyraai/semo
cd 
```

#### 2. 安装依赖

项目使用 `yarn` 来安装依赖包。运行以下命令：

```bash
yarn install
```

#### 3. 配置环境变量

根据项目需求，你可能需要在根目录下创建 `.env` 文件。将 `NEXT_PUBLIC_API_URL` 变量设置为后端 API 的基础 URL。

```bash
NEXT_PUBLIC_API_URL=https://your-backend-url
```

#### 4. 启动开发服务器

使用 `npx expo start` 来启动移动端开发服务器，并通过 Expo Go 应用在移动设备上查看效果：

```bash
# 使用 Expo 启动开发环境
npx expo start
```



### 4. 常见问题

#### 1. 依赖问题
如果遇到与 `babel-loader` 或 `webpack` 相关的兼容性问题，请检查以下配置文件：
- `babel.config.js`
- `next.config.js`

确保 `react-native-web`、`expo` 和其他相关库的版本是兼容的。

#### 2. 依赖问题处理

在运行 `yarn install` 过程中，若出现依赖冲突或兼容性问题，可以通过以下命令强制重新解决依赖关系：

```bash
yarn install --force
```
